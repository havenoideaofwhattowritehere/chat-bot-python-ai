# Інструкція для запуску Flask-додатка з чат-ботом та аналізом сентименту

## Кроки для запуску

1. **Установка необхідних бібліотек**:
   - Переконайтеся, що у вас встановлений Python (версія 3.6 або новіша).
   - Встановіть Flask і Hugging Face Transformers. Ви можете це зробити за допомогою pip. Відкрийте термінал або командний рядок і введіть:

   ```bash
   pip install flask transformers torch random
   ```

2. **Створення структури папок**:
   - Створіть нову папку для вашого проекту (наприклад, `chatbot`).
   - Перейдіть у цю папку:

   ```bash
   cd chatbot
   ```

   - Створіть підпапку для шаблонів:

   ```bash
   mkdir templates
   ```

3. **Створення файлу Python**:
   - Створіть файл з назвою `app.py` у кореневій папці проекту (в `chatbot`).
   - Скопіюйте та вставте код Python, наданий раніше, у `app.py`.

4. **Створення HTML-шаблону**:
   - Створіть файл з назвою `index.html` у папці `templates`.
   - Скопіюйте та вставте HTML-код, наданий раніше, у `index.html`.

5. **Запуск Flask-додатка**:
   - Переконайтеся, що ви все ще в директорії вашого проекту. У терміналі запустіть команду:

   ```bash
   python app.py
   ```

   - Ви повинні побачити повідомлення, що сервер запущений, і щось на кшталт ` * Running on http://127.0.0.1:5000/`.

6. **Перегляд у браузері**:
   - Відкрийте веб-браузер і введіть адресу:

   ```
   http://127.0.0.1:5000/
   ```

   - Ви повинні побачити інтерфейс чат-бота. Введіть своє повідомлення, щоб почати взаємодію з ботом.

## Примітки:
- Переконайтеся, що у вас є доступ до інтернету, оскільки моделі Hugging Face завантажуються з їх серверів при першому запуску.
- Якщо у вас виникають проблеми з встановленням пакетів, перевірте, чи у вас є права адміністратора на вашому комп'ютері або використовуйте віртуальне середовище (наприклад, `venv`).
